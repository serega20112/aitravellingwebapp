# AI Travelling — документация проекта

Добро пожаловать в репозиторий AI Travelling. Это учебно-практический бэкенд-проект, демонстрирующий принципы DDD (Domain-Driven Design) и чистой архитектуры на Python (Flask + SQLAlchemy), с изоляцией домена от инфраструктуры и покрытием тестами.

Основные цели:
- Показывать правильную композицию зависимостей (DI) и использование паттернов Ports & Adapters, Unit of Work.
- Обеспечить хорошую модульность, тестопригодность и наглядность архитектуры для демонстраций.
- Интеграция простого AI-сервиса (через порт), без жёсткой привязки к конкретной библиотеке.

Ключевые технологии:
- Python 3.11+
- Flask, Flask-Login, Flask-SQLAlchemy, Flask-Migrate
- SQLAlchemy 2.x
- Pydantic (валидация входных данных)
- Pytest (юнит и интеграционные тесты)
- Опционально: transformers + torch (для AI)

Полезные материалы:
- docs/architecture.md — архитектура и структура слоёв
- docs/setup.md — установка и запуск (Windows/Linux)
- docs/repository.md — как работать с репозиторием, запуск линтеров и тестов
- docs/testing.md — организация и запуск тестов
- docs/dependencies.md — про зависимости, разделение dev/lint и установка torch
